# Superstore-Sales-Dashboard---Power-BI-Project

Project Overview

This project was completed as part of Week 4 of the AnalystLab Africa Data Analytics Internship.

The objective was to transform raw retail sales data into meaningful business insights using data visualization and dashboarding techniques in Microsoft Power BI.

Dataset

Dataset used: Sample Superstore Dataset

The dataset contains retail transaction records including:

- Orders
- Customers
- Products
- Sales
- Profit
- Discount
- Regions
- Categories
- Order Dates

Business Problem

Raw business data often contains hidden patterns that are difficult to identify through spreadsheets alone.

This project answers key business questions such as:

- Which region generates the highest sales?
- Which region is most profitable?
- Which products perform best?
- How do discounts affect profit?
- What sales trends exist over time?

Key Performance Indicators (KPIs)

- Total Sales: $2.3M
- Total Profit: $286K
- Total Quantity Sold: 38K

Dashboard Features

The dashboard includes:

- KPI Cards
- Sales by Region
- Profit by Region
- Monthly Sales Trend
- Profit by Category
- Discount vs Profit Scatter Plot
- Sales by Sub-Category
- Interactive Filters (Region & Segment)

Key Insights

1. West region generated the highest sales ($725K)
2. West region also produced the highest profit ($108K)
3. Phones were the best-selling sub-category
4. Higher discounts negatively affected profitability
5. Sales showed an upward trend over time

Recommendations

- Increase investment in high-performing regions
- Reduce excessive discounting
- Improve underperforming regions
- Focus on profitable product lines
